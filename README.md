# TPO Management System

## Project Description
This project is a Training and Placement Office (TPO) Management System developed using Python and MySQL. It manages student data, placement details, and company information.

## Technologies Used
- Python
- MySQL
- HTML
- CSS
- XAMPP

## Features
- Student Registration
- Admin Login
- Company Records
- Placement Tracking

## How to Run
1. Start XAMPP (Apache & MySQL)
2. Import database in phpMyAdmin
3. Run the Python project
